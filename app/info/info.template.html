Topsoil-root is a simple kickstarter for AngularJS. It was designed for development of the Harvest webapp.
Topsoil-root uses:
- bower
- npm
- angularjs
- twitter bootstrap
- fontawesome
- jquery
- angular-route
- karma
- protractor

Opinions:
Topsoil-root is remarkably simple. Here are it's opinions:
1. /app contains modules
2. each module x has x.component.js (component), x.component.spec.js (karma tests), x.module.js (module), and x.template.html (template)
3. /assets contains images and other resources
4. /common contains common js
5. /styles contains custom css, less, sass
6. /index.html is the base of our app.

cheers.
